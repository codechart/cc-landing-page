CODE CHART

Visually Navigate Code

Extract the compressed folder Run the executable and go to http://localhost:2900/ in your browser
The executable runs the UI, Agent (searching your code), and Archive (containing saved diagrams)
You can use a shared Archive. Just install CodeChart on a shared machine, and direct you local CodeChart to that machines host using the config.json file 
config.json: archiveUrl
